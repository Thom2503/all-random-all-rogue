# ALL RANDOM ALL ROGUE
A new roguelike, my last one didn't turn out too well, so I decided to start
over on same principles. But the principles have changed a bit, because last
time I wanted it to be kind of greco-roman themed which didn't work out as I
hoped. So this is going to change to a more accessible kind of roguelike with
thight corridor dungeons and predictable but all random enemies and weapons and
dungeon layouts. I will keep the ASCII theme!

## Requirements
I don't want to use any third-party libraries, so the only requirement is Python 3.1x

## Usage
How to use it?
```bash
git clone github.com/Thom2503/all-random-all-rogue.git
```
then
```bash
cd all-random-all-rogue
```
then to actually run it
```bash
python3 main.py
```
